# Kavya-Kanaja

Kavya-Kanaja is an Android-based Kannada poetry learning and exploration application designed to improve regional language learning through interactive and AI-assisted features.

## Features

* Poem of the Day
* Kannada Poetry Learning
* Bhavartha Explanation
* Text-to-Speech Support
* Ask AI Feature
* Poet Information Module
* Interactive User Interface

## Technologies Used

* Android Application Development
* Firebase Integration
* Text-to-Speech APIs
* AI-assisted Response System
* Component-Based Mobile Architecture

## Project Modules

1. Authentication Module
2. Poem Display Module
3. AI Interaction Module
4. Poet Information Module
5. Audio/TTS Module

## Application Screens

* Home Screen

* Audio Learning Screen
